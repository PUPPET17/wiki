# Building Effective AI Agents

Source URL: https://www.anthropic.com/research/building-effective-agents
Author: Anthropic
Date: 2024-12-19
Type: engineering blog
Reliability: high.

Key extracted claims:
- Successful agent systems are often simple, composable, purpose-built.
- Start with the simplest solution and increase complexity only when needed.
- Agentic systems trade performance for latency, cost, complexity, and compounding errors.
- Augmented LLM = LLM + retrieval + tools + memory.
- Relevance: LLM Wiki MVP should avoid over-engineered graph/vector stacks until measured need appears.
