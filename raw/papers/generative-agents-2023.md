# Generative Agents

Source URL: https://arxiv.org/abs/2304.03442
Authors: Joon Sung Park, Joseph C. O'Brien, Carrie J. Cai, Meredith Ringel Morris, Percy Liang, Michael S. Bernstein
Date: 2023-04-07; revised 2023-08-06
Type: paper
Reliability: high.

Key extracted claims:
- Agents store complete experience records in natural language.
- They synthesize memories into higher-level reflections and retrieve dynamically for planning.
- Ablations show observation, planning, and reflection all contribute to believability.
- Relevance: memory lifecycle pattern: observe -> store -> reflect/summarize -> retrieve -> plan/action.
