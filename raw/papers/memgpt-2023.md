# MemGPT: Towards LLMs as Operating Systems

Source URL: https://arxiv.org/abs/2310.08560 and https://research.memgpt.ai/
Authors: Charles Packer, Sarah Wooders, Kevin Lin, Vivian Fang, Shishir G. Patil, Ion Stoica, Joseph E. Gonzalez
Date: 2023-10-12; revised 2024-02-12
Type: paper / project
Reliability: high.

Key extracted claims:
- LLMs are constrained by limited context windows.
- MemGPT proposes virtual context management inspired by OS hierarchical memory.
- It manages information across memory tiers and uses interrupts/control flow.
- Evaluated for large document analysis and multi-session chat.
- Relevance: formalizes agent memory as active memory management, not passive retrieval.
