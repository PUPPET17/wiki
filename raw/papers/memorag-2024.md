# MemoRAG

Source URL: https://arxiv.org/abs/2409.05591
Authors: Hongjin Qian et al.
Date: 2024-09-09; revised 2025-04-09
Type: paper
Reliability: high.

Key extracted claims:
- Conventional RAG assumes explicit queries and well-structured knowledge; long-context tasks often violate both.
- MemoRAG uses a light long-range system to build global memory and draft clues, then an expensive expressive system to answer from retrieved evidence.
- Relevance: supports two-tier architecture: cheap global memory/clue generator plus expensive final synthesis.
