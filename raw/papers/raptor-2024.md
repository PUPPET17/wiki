# RAPTOR

Source URL: https://arxiv.org/abs/2401.18059
Authors: Parth Sarthi et al.; Christopher D. Manning included
Date: 2024-01-31
Type: paper
Reliability: high.

Key extracted claims:
- Standard retrieval often retrieves only short contiguous chunks, limiting holistic document understanding.
- RAPTOR recursively embeds, clusters, and summarizes text into a tree for retrieval at multiple abstraction levels.
- Relevance: wiki pages can be seen as human/agent-readable hierarchical summaries; tree retrieval is a scalable pattern.
