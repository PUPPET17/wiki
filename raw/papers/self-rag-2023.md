# Self-RAG

Source URL: https://arxiv.org/abs/2310.11511
Authors: Akari Asai, Zeqiu Wu, Yizhong Wang, Avirup Sil, Hannaneh Hajishirzi
Date: 2023-10-17
Type: paper
Reliability: high.

Key extracted claims:
- Fixed retrieval can hurt generation when retrieval is unnecessary or irrelevant.
- Self-RAG teaches models to decide when to retrieve and critique evidence/output using reflection tokens.
- Relevance: retrieval in an agent memory system should be adaptive and evidence-aware, not unconditional top-k.
