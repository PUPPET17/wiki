## Key Updates Timeline

### June 3, 2025 Update

> _Memory improvements are starting to roll out for free users. In addition to the saved memories that were there before, ChatGPT now references your recent conversations to provide more personalized responses._

Key points:

- Free users now begin receiving **memory improvements**.
- ChatGPT can reference:
  - **Saved memories**
  - **Recent conversations**
- Free users get a **lightweight version** focused on short-term continuity.
- Plus and Pro users receive memory with a **longer-term understanding** of the user.
- Users can disable memory-related features at any time.

> _You’re in control of ChatGPT’s memory and can turn off referencing “saved memories” or “chat history” at any time in Settings._

---

### April 10, 2025 Update

OpenAI expanded ChatGPT memory to be more comprehensive.

> _Memory in ChatGPT is now more comprehensive. In addition to the saved memories that were there before, it now references all your past conversations to deliver responses that feel more relevant and tailored to you._

Memory now works in two ways:

1. **Saved memories**
   - Things the user explicitly asked ChatGPT to remember.
2. **Chat history**
   - Insights ChatGPT gathers from past conversations to improve future responses.

User control options include:

- Turn off referencing **saved memories**
- Turn off referencing **chat history**
- Ask ChatGPT to change what it knows
- Use **Temporary Chat** for conversations that do not use or update memory
- Ask ChatGPT what it remembers

Availability noted in this update:

- Available to **Plus** and **Pro** users
- Coming to **Team, Enterprise, and Edu** users “in a few weeks”

> _The more you use ChatGPT, the more useful it becomes. New conversations build upon what it already knows about you to make smoother, more tailored interactions over time._

---

### September 5, 2024 Update

> _Memory is now available to ChatGPT Free, Plus, Team, and Enterprise users._

Changes based on earlier testing:

- ChatGPT now notifies users when memories are updated.
- Users can hover over **“Memory updated”** and click **“Manage memories”**.
- Users can review what ChatGPT has picked up.
- Users can forget unwanted memories.
- Memories remain accessible in settings.

## Original Announcement Overview

OpenAI began testing memory in ChatGPT to make future conversations more useful by remembering relevant information across chats.

> _Remembering things you discuss across all chats saves you from having to repeat information and makes future conversations more helpful._

The initial rollout was limited to a small portion of ChatGPT Free and Plus users.

Core promise:

- ChatGPT can remember information across chats.
- Users can explicitly control what is remembered or forgotten.
- Memory can be disabled entirely.

> _You’re in control of ChatGPT’s memory. You can explicitly tell it to remember something, ask it what it remembers, and tell it to forget conversationally or through settings._

## How Memory Works

ChatGPT memory can function in two ways:

- The user can **ask ChatGPT to remember something specific**.
- ChatGPT can **pick up useful details itself** over time.

Memory improves with use and allows ChatGPT to personalize responses without repeated instructions.

### Examples Provided by OpenAI

- If you prefer meeting notes with:
  - Headlines
  - Bullets
  - Action items summarized at the bottom  
  ChatGPT can remember this and use the format automatically.

- If you own a neighborhood coffee shop, ChatGPT can use that context when helping brainstorm social posts.

- If you mention you have a toddler who loves jellyfish, ChatGPT may suggest a birthday card featuring:
  - A jellyfish
  - A party hat

- If you are a kindergarten teacher with 25 students and prefer:
  - 50-minute lessons
  - Follow-up activities  
  ChatGPT can use this when creating lesson plans.

## User Controls for Memory

OpenAI emphasizes that users control ChatGPT memory.

### Turn Memory Off

Users can turn memory off here:

> **Settings > Personalization > Memory**

When memory is off:

- ChatGPT will not create new memories.
- ChatGPT will not use existing memories.

### View, Delete, or Clear Memories

Users can manage memory here:

> **Settings > Personalization > Manage Memory**

Available actions:

- View specific memories
- Delete individual memories
- Clear all memories
- Ask ChatGPT conversationally to forget something

Important limitation:

> _Deleting a chat doesn't erase its memories; you must delete the memory itself._

This means memories are not tied directly to individual chat threads. They evolve separately based on interactions.
