# LLM Wiki / 代理记忆研究

这是一个关于LLM Wiki、Agent Memory、Context Compression、Knowledge Integration的研究型知识库。

## 快速入口

- [维基索引](index.md)
- [架构/维护规范](SCHEMA.md)
- [变更日志](log.md)
- [核心研究框架](concepts/llm-wiki-agent-memory-research-framework.md)

## 主题

这个知识库用于整理和进行以下方向的数据与判断：

- 特工长期记忆
- RAG 与混合搜索
- 上下文工程与上下文压缩
- LLM Wiki / 个人知识库架构
- 知识整合、证据质量与来源追踪
- 可实现的MVP与工程约束

## 结构

- `raw/`：原始资料，包含论文、文章、社区讨论、产品文档
- `concepts/`：整理后的概念页面和综合分析
- `entities/`：实体页
- `comparisons/`：对比分析
- `queries/`：问题与研究查询
- `SCHEMA.md`：wiki结构、frontmatter、标签和更新规则
- `log.md`：追加式维护日志

## 维护原则

- 原始数据保持不可变
-非简单承必须有来源
- 区分事实、推断、猜测和社区观点
- 新证据与旧观点冲突时保留修改记录
