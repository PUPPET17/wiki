Obsidian Web Clipper 是一款免费的浏览器扩展，可让您突出显示页面并将 Web 内容保存到您的保管库。

## 安装网页剪辑器

从您的浏览器的官方目录获取 Obsidian Web Clipper 扩展：

- [Chrome Web Store](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
- 适用于 Chrome、Brave、Arc、Orion 和其他基于 Chromium 的浏览器。
- [Firefox 附加组件](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
	- 适用于 Firefox 和 Firefox Mobile。
- [Safari 扩展](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
	- 适用于 macOS、iOS 和 iPadOS。
- [边缘附加组件](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
	- 对于微软边缘。
